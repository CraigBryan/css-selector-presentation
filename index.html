The * selector

Select EVERYTHING 

<div>
	<p>Some stuff!</p>
</div>

<span>
	<p>Some more stuff!</p>
</span>

* {
    background-color: #DDDDDD;
    color: #222222;
}
						
<div>
	<p>Some stuff!</p>
</div>	

<span>
	<p>Some more stuff!</p>
</span>

The id selector

Select things with id X

<div>
	<p id="awesome">Some stuff!</p>
</div>

<span>
	<p>Some more stuff!</p>
</span>

    background-color: #DDDDDD;
    color: #222222;

Combined selectors

Select things that are X and Y

Syntax: XY

<div class="magic">
    <p>Some stuff!</p>
    <p>Some more stuff!</p>
</div>

<span class="magic">
    <p>More more things</p>
</span>

<div>
    <p class="magic orange">Some repeated stuff!</p>
    <p>Some more repeated stuff!</p>
</div>

div.magic {
    background-color: #DDDDDD;
    color: #222222;
}

.magic.orange {
    background-color: orange;
    color: blue;
}
						
<div class="magic">
    <p>Some stuff!</p>
    <p>Some more stuff!</p>
</div>

<span class="magic">
    <p>More more things</p>
</span>

<div>
    <p class="magic orange">Some repeated stuff!</p>
    <p>Some more repeated stuff!</p>
</div>

Decendants selector

Select children of X that are Y

Syntax: X  Y

    <ul>

        <li>Example</li>
        <li>List</li>
        <li>Ever</li>
    </ul>
