The * selector

Select EVERYTHING

	<p>Some stuff!</p>
</div>

<span>
	<p>Some more stuff!</p>
</span>

    background-color: #DDDDDD;
    color: #222222;
}

Combined selectors

Select things that are X and Y

Syntax: XY

<div class="magic">
    <p>Some stuff!</p>
    <p>Some more stuff!</p>
</div>

<span class="magic">
    <p>More more things</p>
</span>

<div>
    <p class="magic orange">Some repeated stuff!</p>
    <p>Some more repeated stuff!</p>
</div>

div.magic {
    background-color: #DDDDDD;
    color: #222222;
}

.magic.orange {
    background-color: orange;
    color: blue;
}

    <p>Some stuff!</p>

</div>

<span class="magic">

Direct decendants selector

Select direct children of X that are Y

Syntax: X > Y

<div class="outer-parent">
    <div>
        <p>This will not be selected</p>
        <p>Nor will this!</p>
    </div>
</div>

<div class="outer-parent">
    <p>This will be selected!</p>
    <p>And this! Wow.</p>
</div>

.outer-parent > p {
    background-color: #DDDDDD;
    color: #222222;
}
                        
<div class="outer-parent">
    <div>
        <p>This will not be selected</p>
        <p>Nor will this!</p>
    </div>
</div>
        
<div class="outer-parent">
    <p>This will be selected!</p>
    <p>And this! Wow.</p>
</div>

Siblings Selector

Select later siblings of X that are Y

Syntax: X ~ Y

<div id="first-section">
    <p>I totally</p>
    <p>Stole</p>
    <div>
        <p>this example.</p>
    </div>
    <p>Thanks css-tricks.com</p>
</div>

<div id="second-section">
    <p>I totally</p>
    <p>Stole</p>
    <div>
        <p>this example</p>
    </div>
    <p>Thanks css-tricks.com</p>
</div>

#first-section p ~ p {
    background-color: #DDDDDD;
    color: #222222;
}

#second-section div ~ p {
    background-color: orange;
    color: blue;

Attributes selector

Select element type X that have attribute Y

Syntax: X[Y]

Attribute existence

X[Y]


a[title] {
    /* css rules */
}

Example DotCom Link

Attribute exact match

X[Y="exact value"]


a[href="www.craigbryan.me"] {
    /* css rules */
}

My Website

Attribute 'contains' match

X[Y*="contains this"]


a[href*="uottawa"] {
    /* css rules */
}

SESA Website

Attribute 'starts with' match

X[Y^="starts with this"]


a[href^="github"] {
    /* css rules */
}

My GitHub Profile

Attribute 'ends with' match

X[Y$="ends with this"]


a[href$=".gif"] {
    /* css rules */
}
